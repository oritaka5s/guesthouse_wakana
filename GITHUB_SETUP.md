# Airリザーブ 連携ガイド

このガイドでは、Airリザーブとサイトの連携方法を説明します。

## ✅ 現在の状態

**すでに設定済み**：サイトの全予約ボタンがAirリザーブに接続されています。

予約URL: `https://airrsv.net/guesthouse-wakana/calendar`

---

## 🔗 予約ボタンの設置箇所

以下の箇所でAirリザーブに誘導します：

1. ヒーローセクション「今すぐ予約する」
2. ナビゲーション「ご予約」
3. スティッキーCTA「今すぐ予約」（スクロール時に表示）
4. カレンダー「この日程で予約する」

---

## 📧 自動メール設定

### Step 1: Airリザーブでメール設定

1. Airリザーブ管理画面にログイン
2. **設定** → **メール設定**
3. 以下のメールを設定：

#### 予約受付メール（お客様向け）

```
件名: 【ゲストハウス wakana】ご予約ありがとうございます

〇〇 様

この度はゲストハウス wakanaをご予約いただき、
誠にありがとうございます。

【ご予約内容】
チェックイン: {チェックイン日時}
ご宿泊人数: {人数}名
ご予約番号: {予約番号}

【料金】
合計: ¥{料金}

【アクセス】
山形県鶴岡市高田下村25-2
JR鶴岡駅より無料送迎いたします。

ご質問がございましたらお気軽にご連絡ください。

ゲストハウス wakana
TEL: 0235-00-0000
```

---

## 📅 予約済み日程の管理

### サイトのカレンダーを更新する方法

1. Airリザーブで予約が入る
2. `index.html` のJavaScript部分を編集：

```javascript
const bookedDates = [
  '2026-03-15','2026-03-16', // 新しい予約を追加
];
```

3. GitHubにファイルをアップロードして更新

---

## 🎯 運用フロー

### 予約が入ったら

1. Airリザーブから自動メールが送信される
2. 管理画面で予約内容を確認
3. 必要に応じてお客様に連絡
4. サイトのカレンダーに予約済み日を追加

### チェックイン当日

1. 送迎が必要な場合は事前に時間を確認
2. お部屋の準備
3. ウェルカムメッセージ

---

## 💡 便利な機能

### リマインダーメール
チェックイン3日前に自動送信設定可能

### Google カレンダー連携
予約をGoogleカレンダーに自動同期

### LINE通知
予約通知をLINEで受け取る

---

## 🔧 トラブルシューティング

### 予約ボタンが機能しない
- URLが正しいか確認: `https://airrsv.net/guesthouse-wakana/calendar`
- ブラウザのキャッシュをクリア

### メールが届かない
- 迷惑メールフォルダを確認
- Airリザーブのメール設定を確認
